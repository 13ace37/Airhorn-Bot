# Airhorn Bot

A DiscordJS based bot who's playing an airhorn sound on demand.

## Setup

1. rename `example-config.js` into `config.js`
2. type in your token, prefix and wanted play-command.
3. run `npm i && npi start`

## Usage

1. Join a voice channel. (Bot needs to have access to the given channel)
2. Type `[prefix|/]{playCommand|join}` in any for the bot visible, guild text channel.
3. The bot will join in a few seconds and plays the airhorn sound.
4. The bot should leave the channel after the sound is finished.
